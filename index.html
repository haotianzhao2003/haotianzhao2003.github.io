JDK的安装

卸载：在控制面版卸载了旧版本的JDK。在环境变量里删除了有关Java的路径。

去Oracle官网下载了JDK21，安装到了D盘develop文件夹下。（后续又安装了java8，玩游戏用）

在命令行验证了JDK是否安装成功，输入java -version和javac -version查看各自的版本号。

JDK中最重要的2个命令程序是javac和java。javac是“编译代码”的工具，java是用来”执行编译后代码“的工具。

DOS窗口常见命令

D: 切换到D盘

dir 查看当前路径下的信息

cd 进入/回退目录

cls 清屏

开发HelloWorld程序

创建HelloWorld.txt文件，输入一段打印“HelloWorld”的Java代码，再把文件后缀改为.java。

命令行切换到文件所在目录，输入命令javac HelloWorld.java进行编译，之后输入命令java HelloWorld，最终成功输出HelloWorld。

  • 文件名称后缀必须是java
  • 文件名称必须与代码的类名称一致
  • 必须使用英文模式下的符号

JDK的组成

JVM (Java Virtual Machine)：Java虚拟机，真正运行Java程序的地方。

核心类库：Java自己写好的程序，程序员自己的程序可以调用。

JVM + 核心类库 = JRE (Java Runtime Environment)：Java的运行环境。

JRE + 开发工具（Java,Javac…） = JDK (Java Development Kit)：Java开发工具包。

Java的跨平台

一次编译，处处可用，程序只需要开发一次，就可以在各种安装了JVM的系统平台（Windows/Linux/MacOS）上运行。

Path 环境变量

Path 环境变量：用于记住程序路径，方便在命令行窗口的任意目录启动。

JAVA_HOME 环境变量

告诉操作系统JDK安装在了哪个位置（将来其他技术要通过这个环境变量找JDK）。

集成开发环境（IDE, Integrated Development Environment）

把代码编写，编译，执行等多种功能综合到一起的开发工具，可以进行代码智能提示，错误提醒，项目管理等等。

常见的Java IDE工具有：Eclipse，MyEclipse，Intellij IDEA，Jbuilder，NetBeans等。

IDEA管理Java程序的结构

  • project（项目、工程）
  • module（模块）
  • package（包）
  • class（类）

这样划分便于管理项目代码……

在IDEA中创建新项目（空工程）“JavaSE”。在项目中创建module“Day1”。在module中创建package“cn.htzhao.hello”，在package中创建class“HelloWorld”。

IDEA中的Java程序是自动编译和执行的，编译后的class文件在工程路径下的out文件夹里。

导入模块：不要导入文件夹，而是导入文件夹里的.iml文件。万能方法：新建模块，再把src文件夹里的东西复制过来。

删除模块：先在IDEA里Remove模块，再在工程目录里删。

注释

  • 单行注释：// 快捷键 Ctrl + /
  • 多行注释：/* */ 快捷键 Ctrl + Shift + /
  • 文档注释：/** */

字面量

字面量就是告诉程序员：数据在程序中的书写格式。

  • 字符必须用单引号围起来，有且仅有一个字符
  • 字符串必须用双引号围起来

几个常见的特殊值书写格式

数据存储原理

计算机表示数据的最小组成单元是字节（Byte）。

字符存的是ascii码表中对应的数字的二进制形式。’A’对应65，’a’对应97，’0’对应48。

图片和音视频也是采用二进制进行存储。

进制

  • 二进制：0B/0b

  • 八进制：0

  • 十六进制：0X/0x

自动类型转换

类型范围小的变量，可以直接赋值给类型范围大的变量。

表达式的自动类型转换

  • 表达式的最终结果类型由表达式中的最高类型决定
  • 在表达式中，byte、short、char是直接转换成int类型参与运算的

强制类型转换

类型范围大的数据or变量，直接赋值给类型范围小的变量，会报错。

强制类型转换可能会导致数据丢失。

小数强制转换成整数是直接截断小数，保留整数。

算术运算符

除以的符号是“/” ，两个整数相除还是整数，因为最高类型是int。

可以在表达式中*1.0，这样最高类型就是double类型，会保留小数。

“%”是取余数。

“+”可以做连接符。

怎么区分”+”做运算符还是连接符呢？能算则算，不能算放一起。

自增自减

只能对变量操作。

主要关注扩展的赋值运算符，自带强制类型转换。

关系运算符

判断后返回布尔类型的值：true/false。

逻辑运算符

把多个条件放在一起运算，最终返回布尔类型的值：true/false。

符号 叫法 例子 运算逻辑
& 逻辑与 2>1 & 3>2 多个条件都是true，结果才是true。否则为false。
| 逻辑或 2>1 | 3<5 有一个条件是true，结果就是true
！ 逻辑非 ! (2>1) 取反。
^ 逻辑异或 2>1 ^ 3>1 前后条件结果相同，返回false；前后条件结果不同，返回true。
符号 叫法 例子 运算
&& 短路与 2>10 && 3>2 判断结果与”&”一样，过程不同：左边为false，右边则不执行。
|| 短路或 2>1 || 3<5 判断结果与”|”一样，过程不同：左边为true，右边则不执行。

实际开发中，常用的逻辑运算符还是：**&&、||、！**。

三元运算符

计算表达式的结果，结果为true返回值1，结果为false返回值2。

运算符优先级

记住：”( )”的优先级最高，”*”，”/“的优先级高于”+”，”-“

API

回顾之前的知识：JRE = JVM + 核心类库。API算是核心类库中的一部分。

API（Application Programming Interface，应用程序编程接口），就是Java写好的程序，程序员可以直接拿来调用。

Java为自己写好的程序提供了相应的程序使用说明书（API文档）。
