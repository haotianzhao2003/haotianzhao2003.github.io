Java SE | Phase 1

JDK的安装

卸载：在控制面版卸载了旧版本的JDK。在环境变量里删除了有关Java的路径。

去Oracle官网下载了JDK21，安装到了D盘develop文件夹下。（后续又安装了java8，玩游戏用）

在命令行验证了JDK是否安装成功，输入java -version和javac -version查看各自的版本号。

JDK中最重要的2个命令程序是javac和java。javac是“编译代码”的工具，java是用来”执行编译后代码“的工具。

DOS窗口常见命令

D: 切换到D盘

dir 查看当前路径下的信息

cd 进入/回退目录

cls 清屏

开发HelloWorld程序

创建HelloWorld.txt文件，输入一段打印“HelloWorld”的Java代码，再把文件后缀改为.java。

命令行切换到文件所在目录，输入命令javac HelloWorld.java进行编译，之后输入命令java HelloWorld，最终成功输出HelloWorld。

  • 文件名称后缀必须是java
  • 文件名称必须与代码的类名称一致
  • 必须使用英文模式下的符号

JDK的组成

JVM (Java Virtual Machine)：Java虚拟机，真正运行Java程序的地方。

核心类库：Java自己写好的程序，程序员自己的程序可以调用。

JVM + 核心类库 = JRE (Java Runtime Environment)：Java的运行环境。

JRE + 开发工具（Java,Javac…） = JDK (Java Development Kit)：Java开发工具包。

Java的跨平台

一次编译，处处可用，程序只需要开发一次，就可以在各种安装了JVM的系统平台（Windows/Linux/MacOS）上运行。

Path 环境变量

Path 环境变量：用于记住程序路径，方便在命令行窗口的任意目录启动。

JAVA_HOME 环境变量

告诉操作系统JDK安装在了哪个位置（将来其他技术要通过这个环境变量找JDK）。

集成开发环境（IDE, Integrated Development Environment）

把代码编写，编译，执行等多种功能综合到一起的开发工具，可以进行代码智能提示，错误提醒，项目管理等等。

常见的Java IDE工具有：Eclipse，MyEclipse，Intellij IDEA，Jbuilder，NetBeans等。

IDEA管理Java程序的结构

  • project（项目、工程）
  • module（模块）
  • package（包）
  • class（类）

这样划分便于管理项目代码……

在IDEA中创建新项目（空工程）“JavaSE”。在项目中创建module“Day1”。在module中创建package“cn.htzhao.hello”，在package中创建class“HelloWorld”。

IDEA中的Java程序是自动编译和执行的，编译后的class文件在工程路径下的out文件夹里。

导入模块：不要导入文件夹，而是导入文件夹里的.iml文件。万能方法：新建模块，再把src文件夹里的东西复制过来。

删除模块：先在IDEA里Remove模块，再在工程目录里删。
